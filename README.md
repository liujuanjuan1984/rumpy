# rumpy

Python SDK for [QuoRum](https://github.com/rumsystem/quorum).

### About QuoRum

- https://rumsystem.net/
- https://github.com/rumsystem/quorum/blob/main/Tutorial.md
- https://github.com/rumsystem/quorum

### Install

[rumpy@pypi](https://pypi.org/project/rumpy/)

```sh
pip install rumpy
```
